CF 1020D The hat

发表于 2018-08-14

题意

有n($2 <= n <= 1e5, n 为偶数$)个人围成一个圈，每个人有一个身上有一个号码，且相邻两人的号码差值为1，现在需要求是否存在一对对侧的人的号码相同，假如存在，输出下标，假如不存在，输出-1。
最多询问60次，具体交互规则请参考原题。

分析

首先注意到假如对侧两人的差值为奇数，那么一定不存在答案，输出-1。
否则，因为差值是从负数渐变到正数的，所以肯定存在答案，二分即可。
