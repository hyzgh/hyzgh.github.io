2023.09.25周报

发表于 2023-09-24

这周整体比较慵懒，整体只能打6分。理智脑上线得不多，很多时刻是本能脑和情绪脑在上线。还有很多待改进的地方，慢慢来吧。

时间周期：09.18-09.24

生活

  1. 阅读：
    1. 看完了《认知觉醒》，其中一个观点很受启发。阅读的本质是为了改变，阅读 -> 思考 -> 行动 -> 改变。本来想这周写一下读书笔记，但是高估了自己的精力。
    2. 感觉没什么专注力看微信公众号之类的长文，可能大脑被碎片信息搞坏了。慢慢来吧。
  2. 健康：
    1. 打了两场羽毛球。感觉公司的羽毛球局强度不够，俱乐部的强度也参差不齐。打羽毛球也是棋逢对手才好玩。
    2. 三次火影，不好。
    3. 晚上吃零食比较多又不刷牙，对牙齿不好，正在纠正坏习惯。
    4. 开始尝试冥想，有点小收益。
  3. 娱乐：
    1. 看完了《漫长的季节》，不错的一部剧。
    2. 看了三集《人生切割术》，将工作和生活进行记忆上完全分割的想法挺有意思。但是剧情有点慢，没有引起我的兴趣不打算继续追了。
    3. 看了一集《去有风的地方》，感觉刘亦菲演这部剧相对她的其他剧不是那么美，可能她比较适合古装剧。第一集略显悲情和突兀。剧中有一个小装饰品写着亚里士多德的名言：“幸福是人一生追求的最终目的”，引发了我对幸福的一点点思考。
    4. 本来想玩一下游戏，但是发现没什么想玩的，最近switch也没出什么新游，王国之泪不太有玩的兴致了。
  4. 习惯：
    1. 周六去了图书馆，去得比较晚，没有抢到座位也没有用上电脑。但是也学习收获满满，静下心来看了会书。这是最近几个月第一次去图书馆，是个好的开始，继续保持~
    2. 开始培养早睡早起的习惯，慢慢来吧。

技术

  1. 看完了《k8s修炼手册》，但是感觉看过就看过了，没什么深刻的认识，更别谈和自己的知识体系整合了。也打算写写读书笔记，但是高估了自己的精力。
  2. 开始看《凤凰架构》，看完了第一章。感觉作者挺牛的，值得学习。
  3. 工作上，没什么大的突破，感觉在重复之前的经验。另外感觉工作效率不高，一个原因是工作线程开得太多，不够聚焦。慢慢来吧。
  4. AI: 尝试了MiniGPT-4，一个支持图片输入的GPT。尝试的场景是给衣服生成宣传文案，简单试了一下效果一般，可能有改进方法。之前就看过chatGPT即将支持图片输入的信息，有空调研一下。慢慢提升自己使用工具的能力吧~
